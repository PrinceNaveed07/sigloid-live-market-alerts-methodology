# Sigloid Live Market Alerts Methodology

This repository contains the public technical methodology reference for Sigloid Live Market Alerts.

It explains how Sigloid:

- calculates completed 55-day market structure;
- detects bullish and bearish Live Market Alerts;
- sets the Reference Price;
- calculates Current Price deviation;
- assigns and updates Risk Level;
- manages the Alert Lifecycle;
- records alert outcomes for research; and
- sends new-alert notifications.

## Scope

This repository contains public formulas, field definitions, simplified pseudocode, calculation explanations, and methodology limitations.

It does not publish Sigloid’s production bot, private thresholds, credentials, infrastructure, database design, internal services, or proprietary implementation details.

## Research Use

Live Market Alerts identify active changes in completed daily market structure. They are research alerts, not buy or sell signals, financial advice, price predictions, or guaranteed outcomes.
